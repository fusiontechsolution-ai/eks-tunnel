# Release v0.1.0

**Date:** 2026-06-18
**Package:** `@fusiontechsolution.ai/eks-tunnel@0.1.0`
**npm:** https://www.npmjs.com/package/@fusiontechsolution.ai/eks-tunnel

## Summary

Initial public release of eks-tunnel — a globally-installable CLI tool that establishes SSM port-forwarding tunnels to private EKS clusters through bastion hosts.

## Install

```bash
npm install -g @fusiontechsolution.ai/eks-tunnel
```

## Highlights

- **One-command connect** — `eks-tunnel connect my-cluster` handles the entire workflow from credential verification to kubectl configuration
- **Pluggable auth** — supports AWS SSO, static IAM, and external providers (e.g., Opal)
- **Multiple simultaneous tunnels** — each gets its own port and kubectl context
- **Watch mode** — auto-reconnects dropped tunnels with credential refresh
- **Cross-platform** — macOS and Ubuntu/WSL
- **Scriptable** — `--json` and `--quiet` flags, structured exit codes

## Commands

| Command | Description |
|---------|-------------|
| `connect [cluster]` | Connect to a private EKS cluster |
| `status` | List active tunnels |
| `stop <cluster>` | Stop a specific tunnel |
| `stop-all` | Stop all tunnels |
| `watch <cluster>` | Monitor and auto-reconnect a tunnel |
| `refresh <cluster>` | Refresh credentials for a cluster |
| `init` | Create starter config |
| `version` | Show version |

## Configuration

Create `~/.eks-tunnel/clusters.json` (or run `eks-tunnel init`):

```json
{
  "accounts": [
    {
      "accountId": "123456789012",
      "accountName": "my-production",
      "profile": "my-aws-profile",
      "authMethod": "sso",
      "clusters": [
        {
          "name": "eu-west-1-my-cluster",
          "bastionInstanceId": "i-0abc123def456",
          "region": "eu-west-1"
        }
      ]
    }
  ]
}
```

## Known Limitations

- Watch mode auto-reconnect does not persist across CLI restarts (requires `watch` to be running)
- Interactive prompts require a TTY; non-interactive mode exits with helpful error messages
- No Windows native support (use WSL)
