# Release v0.1.3

**Date:** 2026-06-18
**Package:** `@fusiontechsolution.ai/eks-tunnel@0.1.3`
**npm:** https://www.npmjs.com/package/@fusiontechsolution.ai/eks-tunnel

## Summary

Patch release fixing npm Trusted Publishing by upgrading npm CLI to >= 11.5.1 in the publish workflow (required for OIDC authentication). Also fixes dependency vulnerabilities by upgrading vitest.

## Changes

### Fixed

- Publish workflow now explicitly upgrades npm to latest (>= 11.5.1 required for Trusted Publishing OIDC)
- Added version verification step to confirm Node/npm versions before publishing
- Resolved 4 dependency vulnerabilities (2 moderate, 1 high, 1 critical) by upgrading vitest to latest

### Changed

- Upgraded vitest from ^1.6.0 to latest (resolves esbuild/vite security advisories)
- Added `--provenance` flag back to publish command (triggers explicit OIDC detection)

## Install / Upgrade

```bash
npm install -g @fusiontechsolution.ai/eks-tunnel@0.1.3
```
