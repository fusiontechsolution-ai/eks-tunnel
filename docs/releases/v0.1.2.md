# Release v0.1.2

**Date:** 2026-06-18
**Package:** `@fusiontechsolution.ai/eks-tunnel@0.1.2`
**npm:** https://www.npmjs.com/package/@fusiontechsolution.ai/eks-tunnel

## Summary

Patch release fixing the CI/CD publish workflow to use npm Trusted Publishing (OIDC) correctly.

## Changes

### Fixed

- Publish workflow now uses Node 22 (npm Trusted Publishing requires Node >= 22.14.0 and npm CLI >= 11.5.1)
- Removed `--provenance` flag from publish command (provenance is generated automatically with OIDC)
- Removed `NODE_AUTH_TOKEN` — not needed with Trusted Publishing
- Moved `id-token: write` permission to top-level workflow scope

### Added

- `repository.url` field in package.json (required by npm for Trusted Publishing validation)

## Install / Upgrade

```bash
npm install -g @fusiontechsolution.ai/eks-tunnel@0.1.2
```
